# KintoneProcessShow

## Kintone プロセス位置表示プラグイン

## 1.概要

プロセスの現在位置を表示するプラグインです。

表示色は｢済｣｢今｣｢未｣でそれぞれで変更きます。

## 2.注意点

現状αバージョンです。

設定画面のColorPickerで｢OK｣が押せません。

JQueryでcontextが作られない・・・

表示位置はフォーム上部となります。

## 3.今後

cssなど利用し表示の見栄えを良くしようと思います。
